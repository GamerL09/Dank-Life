Setup

  • Grant the bot necessary permissions (Administrator).
  • Use the default prefix: d!
  • Type d!help to see the list of available commands.

📢 Support & Feedback

  • 🔗 Join our Support Server for updates and help.
  • 🐞 Report bugs using: d!report <bug description>
  • 💡 Suggest features with: d!suggest <idea>

Commands

🔨 Moderation Commands

  • d!ban - Bans a user from the server.
  • d!kick - Kicks a user from the server.
  • d!warn - Warns a user.
  • d!mute - Mutes a user.
  • d!unmute - Unmutes a user.
  • d!lock - Locks a channel.
  • d!unlock - Unlocks a channel.
  • d!antilinks - Enables/disables anti-link protection.
  • d!booster - Gives special perks to boosters.
  • d!msg - Sends a message as the bot.
  • d!eping - Sends the event message.
  • d!hping - Sends the hesit message.
  • d!vote - Provides a link to vote for the bot.
  • d!poll - Creates a poll.
  • d!report - Report a user
  • d!suggest - Any suggestions for the bot use this command

🎉 Fun Commands

  • d!tt - Play Tic-Tac-Toe.
  • d!trivia - Play a trivia game.
  • d!kock - Kicks a user (Grinder-only).
  • d!bon - Ban a user (Grinder-only).
  • d!mote - Mute a user (Grinder-only).
  • d!unmote - Unmute a user (Grinder-only).

💰 Currency Commands

  • d!bal - Check your balance.
  • d!work - Work to earn money.
  • d!daily - Claim your daily reward.
  • d!rob - Attempt to rob another user.
  • d!beg - Beg for money.
  • d!bj - Play Blackjack.
  • d!bet - Place a bet.
  • d!with - Withdraw money from your bank.
  • d!dep - Deposit money into your bank.
  • d!pay - Pay another user.
